# ZET Info v3 (Gigahard)

!!! info inline end "Poveznica"
    [https://zet-info.com/](https://zet-info.com/)

Leon Tuček, vlasnik obrta Gigahard koji održava aplikaciju ZET Info,
napravio je nadogradnju aplikacije koristeći
[otvorene podatke ZET-a o](https://www.zet.hr/odredbe/datoteke-u-gtfs-formatu/669)
položaju tramvaja u realnom vremenu. Nadogradnja korisnicima omogućuje da
u svakom trenutku na karti grada vide gdje se nalaze tramvaji koji ih zanimaju,
te dodaje nove funkcionalnosti planiranja putovanja koristeći te podatke.

Aplikacija je dostupna putem trgovina
[Google Play](https://play.google.com/store/apps/details?id=matijakevic.zetinfo)
i
[AppStore](https://apps.apple.com/us/app/zet-info/id1020667929).
